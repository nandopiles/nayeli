API Documentation

This API provides endpoints to manage users. Below are the available endpoints along with their descriptions.

Get All Users

Description: Retrieves all users from the database.

Method: GET

URL: /users

Example Response:

        
            [
                {
                    "id": 1,
                    "username": "User1",
                    "password": "password1"
                },
                {
                    "id": 2,
                    "username": "User2",
                    "password": "password2"
                }
            ]

Get a Specific User

Description: Retrieves a specific user by their ID.

Method: GET

URL: /user

Parameters:

  • id: The ID of the user to retrieve.

Example Response:

        
            {
                "id": 1,
                "username": "User1",
                "password": "password1"
            }

Create a User

Description: Creates a new user with provided username and password.

Method: POST

URL: /user

Request Body: 

        
            {
                "username": "NewUser",
                "password": "newpassword"
            }

Example Response:

        
            {
                "message": "User created",
                "user_id": 3
            }

Delete a User

Description: Deletes a user by their ID.

Method: DELETE

URL: /user

Request Body: 

        
            {
                "id": 3
            }

Example Response:

        
            {
                "message": "User deleted"
            }

Update a User

Description: Updates the username or password of an existing user.

Method: PUT

URL: /user

Request Body: 

        
            {
                "id": 1,
                "username": "UpdatedUser",
                "password": "updatedpassword"
            }

Example Response:

        
            {
                "id": 1,
                "username": "UpdatedUser",
                "password": "updatedpassword"
            }
