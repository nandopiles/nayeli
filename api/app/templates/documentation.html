API Documentation

Get All Users

Description: This method returns all users available in the database.

Method: GET

URL: /users

Example Response:

        
            [
                {
                    "id": 1,
                    "name": "User 1",
                    "age": 25
                },
                {
                    "id": 2,
                    "name": "User 2",
                    "age": 30
                }
            ]

Get a Specific User

Description: This method returns a specific user based on their ID.

Method: GET

URL: /user/{id}

Parameters:

  • id: The ID of the user to be retrieved.

Example Response:

        
            {
                "id": 1,
                "name": "User 1",
                "age": 25
            }
