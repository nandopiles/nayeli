Get a Specific User

Description: Retrieves a specific user by their ID.

Method: GET

URL: /user

Parameters:

  • id: The ID of the user to retrieve.

Request Body:

{
    "id": 3
}

Example Response:

{
    "id": 3,
    "username": "SpecificUser",
    "password": "SpecificPassword",
    "email": "specificuser@example.com",
    "address": "123 Specific St",
    "favs_list": [],
    "bag_list": []
}

Create a User

Description: Creates a new user with provided username and password.

Method: POST

URL: /user

Parameters:

  • username: The name of the user.
  • password: A password.
  • address: The full address of the user.

Request Body:

{
     "username": "NewUser",
     "password": "newpassword",
     "email": "newuser@example.com",
     "address": "123 New St"
}

Example Response:

{
     "message": "User created",
     "user_id": 3
}

Update a User

Description: Updates the username or password of an existing user.

Method: PUT

URL: /user

Request Body:

{
        "id": 1,
        "username": "UpdatedUser",
        "password": "updatedpassword",
        "email": "updateduser@example.com",
        "address": "123 Updated St"
}

Example Response:

{
    "id": 1,
    "username": "UpdatedUser",
    "password": "updatedpassword",
    "email": "updateduser@example.com",
    "address": "123 Updated St",
    "favs_list": [],
    "bag_list": []
}

Delete a User

Description: Deletes a user by their ID.

Method: DELETE

URL: /user

Parameters:

  • id: The ID of the user to retrieve.

Request Body:

{
     "id": 3
}

Example Response:

{
     "message": "User deleted"
}

Get a Specific Product

Description: Retrieves a specific product by its ID.

Method: GET

URL: /product

Parameters:

  • id: The ID of the product to retrieve.

Request Body:

{
    "id": 3
}

Example Response:

{
    "id": 3,
    "name": "SpecificProduct",
    "brand": "SpecificBrand",
    "price": 29.99
}

Update a Product

Description: Updates the name, brand, or price of an existing product.

Method: PUT

URL: /product

Parameters:

  • id: The ID of the product to update.
  • name: The updated name of the product.
  • brand: The updated brand of the product.
  • price: The updated price of the product.

Request Body:

{
    "id": 1,
    "name": "UpdatedProduct",
    "brand": "UpdatedBrand",
    "price": 39.99
}

Example Response:

{
    "id": 1,
    "name": "UpdatedProduct",
    "brand": "UpdatedBrand",
    "price": 39.99
}

Create a Product

Description: Creates a new product with provided name, brand, price.

Method: POST

URL: /product

Parameters:

  • name: The name of the product.
  • brand: The brand of the product.
  • price: The price of the product.

Request Body:

{
    "name": "NewProduct",
    "brand": "NewBrand",
    "price": 29.99
}

Example Response:

{
    "message": "Product created",
    "product_id": 1
}

Delete a Product

Description: Deletes a product by its ID.

Method: DELETE

URL: /product

Parameters:

  • id: The ID of the product to delete.

Request Body:

{
    "id": 3
}

Example Response:

{
    "message": "Product deleted"
}

Search Products

Description: Searches products by name or brand.

Method: POST

URL: /products/search

Parameters:

  • search_type: The type of search. Can be either "name" or "brand".
  • search_term: The term to search for.

Request Body:

{
    "search_type": "name",
    "search_term": "search term here"
}

Example Response:

{
    "results": [
        {
            "id": 1,
            "name": "Product1",
            "brand": "Brand1",
            "price": 99.99
        },
        {
            "id": 2,
            "name": "Product2",
            "brand": "Brand2",
            "price": 79.99
        }
    ]
}
